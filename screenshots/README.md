# Screenshots

Drop PNGs here and reference them from the top-level `README.md`, e.g.:

```markdown
![Agent dashboard](./screenshots/agent-dashboard.png)
```

Suggested set (replace the `> _Screenshot: …_` placeholders in the main README):

| Filename | Shows |

| `anti-leakage-report.png` | Anti-leakage report view |

Tips:
- Use a clean simulator (no debug banners). Hide the Expo dev overlay.
- Crop to the device frame, or use a device-frame tool for polish.
- Redact any real customer names, phone numbers, or plate numbers —
  use the seed/demo data, not production.
- Keep each image under ~500 KB so the page loads fast.
